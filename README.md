# 🎓 Student & Course Management System (Java)

A simple console-based **Student and Course Management System** built using Java.  
This project demonstrates Object-Oriented Programming (OOP) concepts and file handling for persistent data storage.

---

## 📌 Features

### 👨‍🎓 Student Management
- Add new student records
- View all students
- Delete student by name
- Store: ID, Name, Email, Program

### 📚 Course Management
- Add new courses
- View all courses
- Store: Course Code, Course Name, Credit Hours

### 💾 File Storage
- Student data saved in `students.txt`
- Course data saved in `courses.txt`
- Data persists even after program exits

---

## 🧠 OOP Concepts Used

- Abstract Class (`Person`)
- Inheritance (`Student extends Person`)
- Encapsulation (private attributes)
- Method Overriding (`display_info`)
- Object Serialization (manual via strings)
